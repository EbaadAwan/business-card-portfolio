# Business Card Portfolio

This is a solo project that focuses on building user interfaces with React and creating reusable components. The purpose of this mini project was to recreate a personal digital business card while practicing the core concepts of React, including JSX, component-based architecture, and styling.

This project was built as part of Scrimba's Fullstack Developer Path after completing the introductory React module. The goal was to strengthen my understanding of how React applications are structured, how components can be reused to create clean and maintainable code, and how modern frontend applications are developed using Vite.

## Getting Started

1. Clone the repository.

```bash
git clone <repository-url>
```

2. Navigate to the project folder.

```bash
cd business-card-portfolio
```

3. Install dependencies.

```bash
npm install
```

4. Start the development server.

```bash
npm run dev
```

5. Open your browser and visit:

```text
http://localhost:5173
```
